brew下载的mysql开机启动

2018-02-11 mysql mysql 评论

坑1 Path had bad ownership/permissions

可能会有以下提示, 权限不够
Path had bad ownership/permissions

修改用户组可解决
这个plist文件必须是属于root用户，wheel组，用chown修改之

sudo chown root:wheel ~/Library/LaunchAgents/homebrew.mxcl.mysql.plist

坑2 launchctl: no plist was returned

原因是：plist文件内容可能有格式错误。

用下面这个命令可以查看文件格式是否有问题，我就是用这个查看到 有一个<true>标记没有闭合。
